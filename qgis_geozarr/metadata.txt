changelog=
    0.2.0 - Zarr v2 support
    - Parse .zmetadata (v2 consolidated) with zarr.json (v3) fallback
    - Extract CRS from EOPF other_metadata.horizontal_CRS_code
    - Works with EOPF Sentinel-2 L2A production data
    0.1.0 - Initial release
    - STAC browser integration: right-click Zarr assets to load
    - Band/resolution picker with satellite presets (Sentinel-2, Landsat 8/9, MODIS, Sentinel-3)
    - Multiscales convention support (VRT overviews from coarser resolutions)
    - Standalone URL loader with recent URL history
    - Non-blocking metadata fetch
    - Optimized GDAL config for cloud access (HTTP/2, vsicurl cache, parallel decode)
