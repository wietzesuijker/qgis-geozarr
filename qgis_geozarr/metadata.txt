changelog=
    0.1.2 - Thread safety, UX polish, production hardening
    - Fix: thumbnail signal race (connect before thread.start)
    - Fix: thread cleanup on dialog cancel (provider + time series)
    - Fix: STAC cache LRU eviction via OrderedDict
    - Cloud cover color badges (green/amber/red)
    - Band count in group title, file size estimate in resolution dropdown
    - Active preset button highlighting
    - Stretch range persisted per satellite/dtype in QgsSettings
    - HiDPI dialog sizing (540x500)
    - No-results search guidance
    - Consistent error dialogs with expandable detail
    - Pre-warm failure logging
    - Plugin unload robustness (try-finally)
    0.1.1 - Pagination, flat Zarr, Landsat grid support
    - STAC pagination: follows rel=next links (up to 500 items)
    - Flat Zarr stores: arrays without resolution groups now load as single resolution
    - Landsat WRS-2 grid filtering (landsat:wrs_path/wrs_row)
    - User-facing warnings for missing CRS/geotransform metadata
    - Network error logging to QGIS Message Log
    0.1.0 - Initial release
    - Zarr v2 (.zmetadata) and v3 (zarr.json) support
    - STAC browser integration: right-click Zarr assets to load
    - STAC temporal navigator: search by date range, load as time series
    - Grid/tile-aware filtering (grid:code, s2:mgrs_tile) for same-tile series
    - Band/resolution picker with satellite presets (Sentinel-2, Landsat 8/9, MODIS, Sentinel-3)
    - Dataset info header (EPSG, extent, band/resolution counts)
    - Auto RGB styling with cumulative-cut stretch for 3-band composites
    - Band descriptions from metadata in Layers panel
    - Multiscales convention support (VRT overviews from coarser resolutions)
    - EOPF Sentinel-2 support (L2A v2 + Explorer v3, consolidated sub-group CRS)
    - Standalone URL loader with recent URL history
    - Optimized GDAL config for cloud access (HTTP/2, vsicurl cache, parallel decode)
